Focus

基于科学研究的macOS专注计时器，采用随机微休息提醒，提升学习和工作效率

核心功能

⏱️ 专注计时

默认90分钟专注，20分钟休息，可自定义调整时间参数

🔔 微休息提醒

每隔3-5分钟随机提示音，基于神经科学研究的间隔效应原理

📬 系统通知

可开启微休息开始和结束时的系统通知，提供更直观的提醒

🖥️ 全屏黑屏

微休息时自动覆盖所有显示器画面，帮助完全脱离视觉干扰

🎵 自定义音效

支持选择微休息开始和结束的提示音效

🎞️ 视频控制

微休息期间自动暂停/播放视频，观看网课时特别有用

📊 数据统计

可视化图表展示专注数据，支持日/周/月/年多时间段查看

📊 状态栏显示

在macOS菜单栏实时显示剩余时间，支持浅色与深色模式

界面展示

设置界面

现代化设计的设置界面，支持自定义所有参数

数据统计

详细的专注数据分析，帮助了解和改进专注习惯

全屏模式

微休息时的全屏黑屏模式，完全脱离视觉干扰
